## OurFo
ShipShare...galactic bike share (CS320 Project)

## Instructions

# Windows
* Compile with `javac -cp ./postgresql.jar;./ OurFO.java`
* Run with `java -cp ./postgresql.jar;./ OurFO`

# Unix
* Compile with `javac -cp ./postgresql.jar: OurFO.java`
* Run with `java -cp ./postgresql.jar: OurFO`
